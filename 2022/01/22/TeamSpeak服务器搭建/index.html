TeamSpeak服务器搭建

本教程在centos7使用docker来搭建

服务器安全组（出站入站规则）设定，以下这些端口号必须放行（不会的话，请自行baidu）
8767
9987（UDP+TCP）
14534
30033
51234

操作步骤

//先安装docker如果此步骤报错请更新软件包
yum install docker -y
//确认docker安装完毕
docker version
//启动docker
systemctl start docker
//将docker设为开机启动
systemctl enable docker

docker run -dit -p 9987:9987/udp -p 10011:10011 -p 30033:30033 -e TS3SERVER_LICENSE=accept teamspeak

在teamspeak中尝试连接(此时已经能够连接)

现在将获取密钥

docker ps -a    #查看所有容器

找到对应的teamspeak，拷贝对应CONTAINERID

docker logs -f XXXXXX #CONTAINERID

在日志中找到token就可以了

备份密钥

在获取服务器权限以后登录TeamSpeak并连接服务器，在“权限” –> “权限秘钥”创建一个新的服务器组Server Admin的密钥（Privilege Key）并保存好，以备不时之需。
