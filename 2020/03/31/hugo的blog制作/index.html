Hugo的个人博客踩坑事项

个人blog制作的坑，hugo

– 此为本人心酸历程得出，制作环境在manjaro上搭建

  • 安装hugo

    pacman -S hugo

  • 新建hugo博客文件夹

  • hugo new site 文件夹名字*

  • 配置git

  • 如果出现显示你是谁的命令*

–请按照自己的username和email输入

git config –global user.email “你自己的邮箱”

git config –global user.name “用户名”

  • 本地启动预览命令为

  • hugo server -t m10c（m10c为你的主题名） –buildDrafts*

  • 如何新建文章g

–pwd:根目录下
hugo new post/(名字).md
编写是markdown格式

  • 远端部署

    1. 生成public文件夹
      hugo --themes=m10c --baseUrl="https://Dean877.github.io/" --buildDrafts

    2. 进入public文件夹

    3. git init

      git add .

      git commit -m “XXXX”

    4. git remote add origin https://github.com/Dean877/dean877.github.io.git

    5. git push -u origin master

  • 第二次上传

    重新生成public文件夹上传即可

    1. git init

    2. git add .

    3. git commit -m “XXXX”

    4. git remote add origin https://github.com/Dean877/dean877.github.io.git

    5. git status

    6. git push

  • m10c的配色调整

    darkestColor = “#XXX”左边颜色

    darkColor = “#XXX”右边颜色

    lightColor = “#XXX”简介颜色

    lightestColor = “#XXX” 名字颜色

    primaryColor = “#XXX” 头像框色
