list和字符串相互转换

将一个字符串转为一个List

#定义一个字符串名为phrase
phrase = "Hello word!"
#用'list'转换并赋值给plist
plist = list(phrase)

将一个List转为一个字符串

#定义名为plist的List
plist = ['H', 'e', 'l', 'l', 'o', ' ', 'w', 'o', 'r', 'd', '!']
#用join()函数转换
phrase = ''.join(plist)

循环字典

>>> found
{'a': 0, 'e': 0, 'i': 0, 'o': 0, 'u': 0}
>>> for kv in found:
	print(kv)
a
e
i
o
u
#这样循环只能循环出键名
>>> for k in found:
	print(k,found[k])	
a 0
b 0
c 0
d 0
#可以通过这样循环来得到键名和值
#因为字典是无序的，所以for循环出来通常也是无序的排列！！！若想让字典有序排列可以使用sorted函数
#sorted()默认是对字典的键，从小到大进行排序
>>> for k in sorted(found):
    print(k,found[k])
